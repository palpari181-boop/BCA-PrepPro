Chapter 1: Introduction to C Language

C is a powerful, general-purpose programming language developed by Dennis Ritchie in 1972 at Bell Labs. It is one of the most widely used programming languages and forms the foundation of many modern languages like C++, Java, and Python.

C language is known for its simplicity, efficiency, and flexibility. It is used for system programming, application development, embedded systems, and operating systems.

Definition:
C is a structured, procedural programming language used to develop efficient programs with low-level memory access.

Features of C Language:
✔ Simple and easy to learn
✔ Fast execution speed
✔ Portable (machine independent)
✔ Structured programming language
✔ Supports low-level memory manipulation using pointers
✔ Rich set of built-in functions

Why C is called Middle-Level Language?
C is called a middle-level language because it combines features of both high-level languages (like easy syntax) and low-level languages (like memory access).

Applications of C:
• Operating Systems (Unix, Linux)
• Embedded Systems (cars, machines)
• Compilers and interpreters
• Game development
• Database systems

👉 Important: C is the base of almost all programming languages, so it is very important for BCA students.

Chapter 2: Structure of a C Program

A C program is a set of instructions written in a specific structure that compiler can understand. Every C program must contain a main() function.

General Structure of C Program:
1. Documentation Section (comments)
2. Link Section (#include)
3. Definition Section (#define)
4. Global Declaration
5. main() Function
6. Sub Functions

Example Program:

#include <stdio.h>
int main() {
  printf("Hello World");
  return 0;
}

Explanation:
• #include <stdio.h> → standard input/output library
• main() → execution starts here
• printf() → displays output on screen
• return 0 → program ends successfully

Chapter 3: Tokens in C Language

Tokens are the smallest units in a C program. A program is made up of different tokens.

Types of Tokens:
✔ Keywords
✔ Identifiers
✔ Constants
✔ Strings
✔ Operators

1. Keywords:
Keywords are reserved words in C language that have predefined meaning.
Example: int, float, if, else, return, while

2. Identifiers:
Identifiers are names given to variables, functions, arrays etc. Rules:
• Must start with alphabet or underscore
• Cannot use keywords
• Case sensitive

3. Constants:
Fixed values that do not change during program execution. Example: 10, 3.14, 'A'

4. Operators:
Symbols used to perform operations like +, -, *, /, %

Chapter 4: Variables and Data Types (Detailed)

A variable is a named memory location used to store data that can be changed during program execution. Each variable in C must have a specific data type.

Rules for Naming Variables:
✔ Must start with alphabet or underscore (_)
✔ Cannot start with digit
✔ Cannot use keywords (like int, float)
✔ Case sensitive (A and a are different)
✔ No special characters allowed except underscore

Definition Example:
int a = 10;
float b = 5.5;
char c = 'A';

Data Types in C:

1. int (Integer):
Used to store whole numbers.
Example: 10, -5, 100
Memory: 2 or 4 bytes (depends on system)

2. float:
Used to store decimal numbers.
Example: 3.14, 2.5
Memory: 4 bytes

3. double:
Used for large decimal values with higher precision.
Memory: 8 bytes

4. char:
Used to store single character.
Example: 'A', 'b', '9'
Memory: 1 byte

Why Data Types are Important?
Data types tell the compiler how much memory to allocate and what type of data will be stored.

Type Conversion:
• Implicit: Automatically done by compiler
• Explicit: Manually done by programmer using type casting

Example:
int a = 10;
float b = (float)a;

Chapter 5: Input and Output Functions

C language uses standard input and output functions to interact with the user.

1. printf() Function:
Used to display output on screen.

Example:
printf("Hello World");

2. scanf() Function:
Used to take input from user.

Example:
int a;
scanf("%d", &a);

Important Format Specifiers:

✔ %d → integer
✔ %f → float
✔ %lf → double
✔ %c → character
✔ %s → string

Example Program:

int a;
float b;
char c;

scanf("%d %f %c", &a, &b, &c);
printf("%d %f %c", a, b, c);

Important Points:
✔ & is used in scanf to give address
✔ printf does not need &
✔ Format specifier must match data type

Chapter 6: Operators in C (Detailed)

Operators are special symbols used to perform operations on variables and values. C language provides many types of operators used for arithmetic, comparison, logic and assignment.

Types of Operators in C:

1. Arithmetic Operators:
Used for mathematical calculations.
+ (Addition), - (Subtraction), * (Multiplication), / (Division), % (Modulus)

Example:
int a = 10, b = 5;
a + b = 15
a - b = 5

2. Relational Operators:
Used to compare two values. Result is always true (1) or false (0).
==, !=, >, <, >=, <=

Example:
10 > 5 → true (1)
10 == 5 → false (0)

3. Logical Operators:
Used to combine multiple conditions.
&& (AND), || (OR), ! (NOT)

Example:
(10 > 5 && 5 < 2) → false

4. Assignment Operators:
Used to assign values.
=, +=, -=, *=, /=, %=

Example:
a += 5 → a = a + 5

5. Increment/Decrement Operators:
++ (increase by 1), -- (decrease by 1)

Example:
a++ → post increment
++a → pre increment

Important Note:
Operators are very important for decision making and loops in C programming.

Chapter 7: Control Statements in C

Control statements are used to control the flow of execution of a program. They decide which part of code will execute and when.

Types of Control Statements:

1. Conditional Statements:
Used for decision making.

if statement:
Executes block if condition is true.

Example:
if(a > b) {
  printf("A is greater");
}

if-else statement:
Executes one block if true, another if false.

Nested if:
if inside another if.

switch statement:
Used when multiple conditions are present.

Example:
switch(choice) {
case 1: printf("One"); break;
case 2: printf("Two"); break;
default: printf("Invalid");
}

2. Looping Statements:
Used to repeat a block of code multiple times.

Types of Loops:

1. for loop:
Used when number of iterations is known.

Example:
for(int i=1; i<=5; i++) {
  printf("%d", i);
}

2. while loop:
Entry-controlled loop (condition checked first).

3. do-while loop:
Exit-controlled loop (runs at least once).

Example:
do {
  printf("Hello");
} while(condition);

Important Difference:
• for loop → fixed iteration
• while loop → condition based
• do-while → executes at least once

Chapter 8: Functions in C (Detailed)

A function is a block of code that performs a specific task and can be reused multiple times in a program. Functions help in reducing code repetition and improving program structure.

Types of Functions:

1. Library Functions:
These are pre-defined functions provided by C library.
Example: printf(), scanf(), strlen(), sqrt()

2. User-defined Functions:
Functions created by the programmer according to requirement.

Structure of Function:
return_type function_name(parameters) {
  // body of function
}

Example:
int add(int a, int b) {
  return a + b;
}

Advantages of Functions:
✔ Code reusability
✔ Easy debugging
✔ Better program structure
✔ Saves time and effort

Types based on arguments and return value:

1. No argument, no return value
2. No argument, return value
3. Argument, no return value
4. Argument, return value

Recursion:
When a function calls itself is called recursion.

Example:
factorial(n) = n * factorial(n-1)

Chapter 9: Arrays in C

An array is a collection of similar data elements stored in continuous memory locations. It is used to store multiple values in a single variable.

Why Arrays are used?
Instead of creating multiple variables, arrays help store data efficiently.

Types of Arrays:

1. One Dimensional Array (1D):
Used to store list of elements.

Example:
int a[5] = {10, 20, 30, 40, 50};

2. Two Dimensional Array (2D):
Used to store data in matrix form.

Example:
int a[2][2] = { {1,2}, {3,4} };

Advantages of Arrays:
✔ Easy to store multiple values
✔ Easy to access elements using index
✔ Efficient memory usage

Chapter 10: Strings in C (Basic)

A string is a collection of characters terminated by a null character '\0'.

Example:
char name[] = "BCA";

String Functions:

• strlen() → length of string
• strcpy() → copy string
• strcat() → concatenate strings
• strcmp() → compare strings

Example:
char a[10] = "Hello";
char b[10] = "World";
strcat(a, b);

Important Note:
Strings always end with '\0' null character.

Chapter 11: Pointers in C (Detailed)

A pointer is a variable that stores the memory address of another variable. Pointers are one of the most powerful features of C language because they allow direct memory access.

Why Pointers are used?
✔ To access memory directly
✔ To improve program efficiency
✔ Used in dynamic memory allocation
✔ Used in arrays and functions

Pointer Declaration:
datatype *pointer_name;

Example:
int a = 10;
int *p = &a;

Here:
• &a → gives address of variable a
• *p → gives value at that address

Important Pointer Operators:
• & → Address operator
• * → Value at address (dereference operator)

Example Program:
int a = 10;
int *p = &a;
printf("%d", *p);

Advantages of Pointers:
✔ Faster execution
✔ Memory efficiency
✔ Used in data structures
✔ Dynamic memory handling

Disadvantages:
• Complex to understand
• Risk of memory errors

Chapter 12: File Handling in C (Basic)

File handling is used to store data permanently in files on a computer. Unlike variables, file data remains stored even after program ends.

Types of Files:
• Text Files (.txt)
• Binary Files (.bin)

File Operations:
✔ Create a file
✔ Open a file
✔ Read from file
✔ Write to file
✔ Close a file

File Pointer:
FILE *fp;

Common Functions:

• fopen() → open file
• fclose() → close file
• fprintf() → write to file
• fscanf() → read from file
• fgetc() → read character
• fputc() → write character

Example:
FILE *fp;
fp = fopen("data.txt", "w");
fprintf(fp, "Hello World");
fclose(fp);
