🔢 Chapter 1 : Number System & Binary Codes

📖 Introduction

A Number System is a method of representing numbers using a set of digits. In Digital Electronics and Computer Systems, all information is stored and processed in the form of binary digits (0 and 1). Different number systems such as Decimal, Binary, Octal and Hexadecimal are used for calculations, programming and data representation.

📚 Definition

A Number System is a mathematical technique used to represent numbers with the help of specific symbols or digits. Each number system has a fixed base (radix) that determines the number of digits used.

⭐ Types of Number Systems

Number System Base Digits Used
Decimal 10 0 – 9
Binary 2 0, 1
Octal 8 0 – 7
Hexadecimal 16 0 – 9, A – F

🔟 Decimal Number System

The Decimal Number System is the most commonly used number system in daily life. It has a base of 10 and uses ten digits from 0 to 9.

Example: 


45810

💻 Binary Number System

The Binary Number System has a base of 2 and uses only two digits: 0 and 1. Computers understand and process data only in binary form.

Example: 


101101₂

🔶 Octal Number System

The Octal Number System has a base of 8 and uses digits from 0 to 7. It provides a shorter representation of binary numbers.

Example: 


725₈

🟣 Hexadecimal Number System

The Hexadecimal Number System has a base of 16 and uses digits 0–9 and letters A–F. It is widely used in computer programming, memory addressing and web color codes.

Example: 


3AF₁₆

🔄 Number System Conversion

Conversion means changing a number from one number system to another. The most common conversions are Decimal to Binary, Binary to Decimal, Binary to Octal and Binary to Hexadecimal.

Conversion Method
Decimal → Binary Repeated division by 2
Binary → Decimal Positional weights (powers of 2)
Binary → Octal Group bits in sets of 3
Binary → Hexadecimal Group bits in sets of 4

➕ Binary Arithmetic

Binary arithmetic performs mathematical operations using binary digits. The basic operations are Binary Addition, Subtraction, Multiplication and Division.

Example: 


1010
+0101
------
1111

🔄 1's Complement & 2's Complement

Complements are used to represent negative numbers and simplify subtraction operations in digital systems.

Complement Description
1's Complement Invert all bits (0 ↔ 1)
2's Complement 1's Complement + 1
Example: 


Binary : 1010

1's Complement : 0101

2's Complement : 0110

🔤 Binary Codes

Binary Codes are used to represent numbers, letters and special characters in binary form. They improve communication between computers and digital devices.

Code Purpose
BCD Represents decimal digits in binary.
Gray Code Only one bit changes between consecutive values.
ASCII Represents characters in computers.
Unicode Supports multiple international languages.

⭐ Advantages of Binary Number System

  • ✔ Easy for digital circuits.
  • ✔ High accuracy and reliability.
  • ✔ Simple electronic implementation.
  • ✔ Fast data processing.
  • ✔ Used in all modern computers.

❌ Disadvantages

  • ❌ Binary numbers become lengthy.
  • ❌ Difficult for humans to read directly.
  • ❌ Manual calculations are time-consuming.

⚙️ Chapter 2 : Logic Gates

📖 Introduction

Logic Gates are the basic building blocks of digital electronics. They perform logical operations on one or more binary inputs (0 or 1) and produce a single binary output. Every digital circuit, such as computers, calculators, mobile phones and communication systems, is built using logic gates.

📚 Definition

A Logic Gate is an electronic circuit that performs a logical operation based on Boolean Algebra. It accepts binary inputs and produces a binary output according to a specific logical rule.

⭐ Features of Logic Gates

  • ✔ Operate using binary values (0 and 1).
  • ✔ Used in all digital electronic devices.
  • ✔ Perform logical operations quickly.
  • ✔ Easy to combine for complex circuits.
  • ✔ Form the basis of digital computers.

🔷 Types of Logic Gates

OR ≥1 Output is 1 if any input is 1.

XNOR ⊙ Output is 1 when inputs are the same.

The NAND Gate is the inverse of the AND Gate. It gives LOW output only when all inputs are HIGH. It is known as a Universal Gate.

The NOR Gate is the inverse of the OR Gate. It produces HIGH output only when all inputs are LOW. It is also a Universal Gate.

🧮 Chapter 3 : Boolean Algebra & K-Map

📖 Introduction

Boolean Algebra is a branch of mathematics used in digital electronics to perform logical operations using only two values, 0 (False) and 1 (True). It helps in designing and simplifying digital circuits. Boolean Algebra, together with Karnaugh Maps (K-Maps), reduces complex logical expressions, making circuits faster, smaller and more efficient.

📚 Definition

Boolean Algebra is a mathematical system developed by George Boole. It uses binary variables and logical operators such as AND, OR and NOT to represent and simplify digital logic circuits.

⭐ Features of Boolean Algebra

  • ✔ Uses only two values: 0 and 1.
  • ✔ Simplifies logical expressions.
  • ✔ Reduces hardware complexity.
  • ✔ Improves circuit performance.
  • ✔ Widely used in digital electronics and computers.

🔷 Basic Boolean Operations

Operation Symbol Description
AND . Output is 1 only if all inputs are 1.
OR + Output is 1 if any input is 1.
NOT ' Produces the opposite value.

📖 Boolean Laws

Boolean Laws are rules used to simplify logical expressions and design efficient digital circuits.

Law Expression
Identity Law A + 0 = A    A · 1 = A
Null Law A + 1 = 1    A · 0 = 0
Idempotent Law A + A = A    A · A = A
Complement Law A + A' = 1    A · A' = 0
Commutative Law A + B = B + A
Associative Law (A+B)+C = A+(B+C)
Distributive Law A(B+C)=AB+AC

🔄 De Morgan's Theorem

De Morgan's Theorem is used to simplify Boolean expressions and convert AND operations into OR operations (and vice versa) by complementing each variable. It is widely used in logic circuit design.

Theorem 1 


(A + B)' = A' · B'
Theorem 2 


(A · B)' = A' + B'

📌 SOP & POS Forms

Boolean expressions can be represented in two standard forms.

Form Meaning
SOP Sum of Products (AND terms joined by OR)
POS Product of Sums (OR terms joined by AND)
Example 


SOP : AB + A'C

POS : (A+B)(A'+C)

🗂 Karnaugh Map (K-Map)

A Karnaugh Map (K-Map) is a graphical method used to simplify Boolean expressions. It groups adjacent cells containing 1s to produce the simplest possible logic expression, reducing the number of logic gates required.

Variables Cells
2 Variables 4 Cells
3 Variables 8 Cells
4 Variables 16 Cells

⭐ Advantages of K-Map

  • ✔ Simplifies Boolean expressions.
  • ✔ Reduces the number of logic gates.
  • ✔ Minimizes circuit cost.
  • ✔ Increases circuit efficiency.
  • ✔ Easy for small and medium circuits.

💻 Applications of Boolean Algebra

  • ✔ Digital Circuit Design
  • ✔ Computer Processors
  • ✔ Logic Gate Implementation
  • ✔ Control Systems
  • ✔ Communication Systems
  • ✔ Error Detection Circuits

⚡ Chapter 4 : Combinational Circuits

📖 Introduction

A Combinational Circuit is a digital circuit whose output depends only on the present input values. It does not store any previous data or memory. These circuits are widely used in digital systems for performing arithmetic operations, data selection, encoding and decoding.

📚 Definition

A Combinational Circuit is a collection of logic gates connected together to produce a specific output based only on the current combination of inputs. There is no memory element or feedback path in these circuits.

⭐ Features of Combinational Circuits

  • ✔ Output depends only on present inputs.
  • ✔ No memory storage.
  • ✔ No feedback path.
  • ✔ Fast operation.
  • ✔ Easy to design and implement.

➕ Half Adder

A Half Adder is a combinational circuit used to add two binary digits. It produces two outputs: Sum (S) and Carry (C).

Input A Input B Sum Carry
0000
0110
1010
1101
Boolean Expressions 


Sum = A ⊕ B

Carry = A · B

➕ Full Adder

A Full Adder adds three binary inputs (A, B and Carry-In). It produces a Sum output and a Carry-Out output. 



Sum = A ⊕ B ⊕ Cin

Carry = AB + BCin + ACin

➖ Half Subtractor

A Half Subtractor subtracts one binary digit from another. It produces two outputs: Difference and Borrow. 



Difference = A ⊕ B

Borrow = A'B

➖ Full Subtractor

A Full Subtractor subtracts three binary inputs including Borrow-In. It generates Difference and Borrow-Out outputs. 



Difference = A ⊕ B ⊕ Bin

Borrow = A'B + BBin + A'Bin

🔀 Multiplexer (MUX)

A Multiplexer is a data selector. It selects one input from many inputs and sends it to a single output based on the selection lines.

Component Function
Inputs Multiple
Output Single
Select Lines Select one input

🔁 Demultiplexer (DEMUX)

A Demultiplexer performs the opposite operation of a Multiplexer. It accepts one input and distributes it to one of many outputs according to the select lines.

🔢 Encoder

An Encoder converts multiple input lines into a smaller number of binary output lines. It is mainly used for data compression and coding.

Example: 8-to-3 Encoder

🔓 Decoder

A Decoder performs the opposite operation of an Encoder. It converts binary input data into multiple output lines. Decoders are used in memory systems and digital displays.

Example: 3-to-8 Decoder

🌍 Applications of Combinational Circuits

  • ✔ Simple circuit design.

❌ Disadvantages

  • ❌ Cannot store previous data.

⏱️ Chapter 5 : Sequential Circuits

📖 Introduction

Sequential Circuits are digital circuits whose output depends on both the present inputs and the previous state (stored data). Unlike combinational circuits, sequential circuits contain memory elements such as flip-flops. They are widely used in registers, counters, memory devices and digital control systems.

📚 Definition

A Sequential Circuit is a digital circuit that stores previous information and produces output based on both current inputs and stored data. These circuits use clock signals for synchronization.

⭐ Features of Sequential Circuits

  • ✔ Output depends on present and previous inputs.
  • ✔ Contains memory elements.
  • ✔ Uses clock pulses for operation.
  • ✔ Suitable for data storage.
  • ✔ Used in digital systems and processors.

🔄 Flip-Flops

A Flip-Flop is the basic memory element in digital electronics. It stores one binary bit (0 or 1) and changes its state according to the clock signal.

Flip-Flop Description
SR Flip-Flop Basic Set-Reset memory circuit.
JK Flip-Flop Improved version of SR Flip-Flop.
D Flip-Flop Stores one data bit.
T Flip-Flop Toggles its state on each clock pulse.

🟢 SR Flip-Flop

The SR (Set-Reset) Flip-Flop is the simplest flip-flop. It has two inputs, Set (S) and Reset (R), and stores one bit of data.

Functions:
  • S = 1 → Set Output
  • R = 1 → Reset Output
  • S = 0 & R = 0 → No Change
  • S = 1 & R = 1 → Invalid State

🔵 JK Flip-Flop

The JK Flip-Flop removes the invalid state of the SR Flip-Flop. When both inputs are HIGH, the output toggles between 0 and 1.

Special Feature: No Invalid State

🟡 D Flip-Flop

The D (Data) Flip-Flop stores one bit of information and transfers the input to the output on the active clock edge. It is widely used in registers and memory.

🟣 T Flip-Flop

The T (Toggle) Flip-Flop changes its output state whenever the clock pulse arrives. It is mainly used in counters.

💾 Registers

A Register is a group of flip-flops used to store binary data temporarily. Registers are important components of CPUs and digital systems.

Register Type Purpose
Shift Register Shifts data left or right.
Storage Register Stores binary information.

🔀 Shift Registers

Shift Registers are sequential circuits that shift binary data from one flip-flop to another using clock pulses.

Type Description
SISO Serial In – Serial Out
SIPO Serial In – Parallel Out
PISO Parallel In – Serial Out
PIPO Parallel In – Parallel Out

⏲ Counters

Counters are sequential circuits that count the number of clock pulses. They are widely used in timers, digital clocks and frequency counters.

Counter Purpose
Asynchronous Counter Clock is applied only to the first flip-flop.
Synchronous Counter Clock is applied to all flip-flops together.

🌍 Applications of Sequential Circuits

  • ✔ Digital Clocks
  • ✔ Traffic Light Controllers

  • ✔ Supports sequential operations.
