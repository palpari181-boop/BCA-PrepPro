🔢 Chapter 2 : Variables, Data Types & Operators

📖 Introduction

Variables are used to store data in Python programs. Every variable has a name and holds a value that can be changed during program execution. Python supports different data types such as numbers, strings, lists and boolean values. Operators perform mathematical, comparison and logical operations on data, making programming easier and more efficient.

📚 Variable

A Variable is a named memory location used to store data. In Python, variables are created automatically when a value is assigned. No explicit declaration is required.

Example

```python name = "Pari" age = 20 print(name) print(age) ```

📝 Rules for Naming Variables

  • ✔ Variable names must start with a letter or underscore (_).
  • ✔ They cannot start with a number.
  • ✔ Only letters, digits and underscores are allowed.
  • ✔ Python keywords cannot be used as variable names.
  • ✔ Variable names are case-sensitive.

🔑 Keywords

Keywords are reserved words in Python that have predefined meanings and cannot be used as variable names.

Examples

```python if else for while True False class return ```

🏷 Identifiers

Identifiers are the names given to variables, functions, classes and other program elements. They help identify different objects in a program.

📦 Data Types in Python

Data Type Description Example
int Stores integer values. 100
float Stores decimal values. 45.75
str Stores text or characters. "Python"
bool Stores True or False values. True
list Stores multiple values in a sequence. [10,20,30]
tuple Stores ordered immutable values. (1,2,3)
dict Stores data in key-value pairs. {"name":"Pari"}
set Stores unique values. {1,2,3}

🔄 Type Casting

Type Casting is the process of converting one data type into another using built-in functions.

Example

```python age = "20" print(int(age)) print(float(age)) ```

➕ Operators in Python

Operator Type Examples
Arithmetic +, -, *, /, %, //, **
Comparison ==, !=, >, <, >=, <=
Logical and, or, not
Assignment =, +=, -=, *=, /=
Membership in, not in
Identity is, is not

💻 Example Program

```python a = 20 b = 5 print("Addition =", a + b) print("Subtraction =", a - b) print("Multiplication =", a * b) print("Division =", a / b) ```

⭐ Advantages

  • ✔ Variables make programs flexible.
  • ✔ Data types organize different kinds of information.
  • ✔ Operators simplify calculations and comparisons.
  • ✔ Type casting allows easy data conversion.
  • ✔ Easy to write and understand.

❌ Disadvantages

  • ❌ Incorrect data types may produce runtime errors.
  • ❌ Dynamic typing may lead to unexpected results.
  • ❌ Improper variable naming reduces code readability.
  • ❌ Wrong operator usage may produce incorrect output.

⚙️ Chapter 4 : Functions & Modules

📖 Introduction

Functions and Modules are essential concepts in Python programming that help organize code into reusable and manageable parts. Functions perform specific tasks, while modules contain collections of related functions and variables. Using functions and modules makes programs easier to understand, maintain and reuse.

📚 Definition of Function

A Function is a block of reusable code that performs a specific task. Functions help reduce code duplication and improve program readability.

⭐ Types of Functions

Type Description
Built-in Function Functions already provided by Python, such as print() and len().
User-defined Function Functions created by programmers using the def keyword.

🖥 Creating a Function

```python def greet(): print("Welcome to Python") greet() ```

📥 Function with Parameters

```python def add(a, b): print("Sum =", a + b) add(10, 20) ```

📤 Function with Return Value

```python def square(num): return num * num result = square(5) print(result) ```

🌍 Variable Scope

Variable Scope defines where a variable can be accessed within a program. Python mainly supports two types of scope.

Scope Description
Local Variable Accessible only inside the function where it is declared.
Global Variable Accessible throughout the program unless modified locally.

⚡ Lambda Function

A Lambda Function is an anonymous function written in a single line using the lambda keyword. It is mainly used for short operations.

```python square = lambda x: x * x print(square(6)) ```

📦 Modules

A Module is a Python file containing functions, variables and classes that can be imported and reused in other programs. Modules help organize code efficiently.

Example

```python import math print(math.sqrt(25)) print(math.pi) ```

🛠 Common Python Modules

Module Purpose
math Performs mathematical calculations.
random Generates random numbers.
datetime Works with dates and time.
os Interacts with the operating system.
sys Provides system-related functions.

⭐ Advantages of Functions & Modules

  • ✔ Reduces code duplication.
  • ✔ Improves code readability.
  • ✔ Makes debugging easier.
  • ✔ Encourages code reusability.
  • ✔ Simplifies large programs.

❌ Disadvantages

  • ❌ Excessive function calls may reduce performance.
  • ❌ Improper module organization can create confusion.
  • ❌ Global variables may lead to unexpected behavior.
  • ❌ Beginners may find scope management difficult.

📂 Chapter 5 : Lists, Tuples, Dictionaries & Sets

📖 Introduction

Python provides several built-in data structures to store and manage data efficiently. The most commonly used data structures are Lists, Tuples, Dictionaries and Sets. Each has unique features and is used for different types of programming tasks. These data structures make data storage, retrieval and manipulation simple and efficient.

📚 List

A List is an ordered and mutable collection of elements. It allows duplicate values and supports different data types in a single collection.

Example

```python fruits = ["Apple", "Mango", "Orange"] print(fruits) print(fruits[1]) ```

📚 Tuple

A Tuple is an ordered but immutable collection of elements. Once created, its values cannot be changed.

Example

```python numbers = (10, 20, 30) print(numbers) print(numbers[0]) ```

📚 Dictionary

A Dictionary stores data in the form of key-value pairs. Each key must be unique and is used to access its corresponding value.

Example

```python student = { "Name": "Pari", "Age": 20, "Course": "BCA" } print(student["Name"]) ```

📚 Set

A Set is an unordered collection of unique elements. It automatically removes duplicate values and is useful for mathematical set operations.

Example

```python numbers = {10, 20, 30, 20, 10} print(numbers) ```

📁 Chapter 6 : File Handling & Exception Handling

📖 Introduction

File Handling and Exception Handling are important concepts in Python programming. File Handling allows programs to create, read, write and update files, while Exception Handling helps manage runtime errors without stopping program execution. These concepts make Python applications more reliable, secure and user-friendly.

📚 File Handling

File Handling is the process of performing operations such as creating, opening, reading, writing and closing files. Python provides built-in functions to work with text and binary files efficiently.

📂 File Modes

Mode Description
r Read an existing file.
w Write data (creates a new file or overwrites an existing one).
a Append data to the end of a file.
x Create a new file; generates an error if it already exists.
rb Read a binary file.
wb Write to a binary file.

🖥 Reading a File

```python file = open("data.txt", "r") content = file.read() print(content) file.close() ```

📝 Writing to a File

```python file = open("data.txt", "w") file.write("Welcome to Python") file.close() ```

➕ Appending to a File

```python file = open("data.txt", "a") file.write("\nLearning File Handling") file.close() ```

⚠ Exception Handling

Exception Handling is used to handle runtime errors and prevent a program from crashing. Python uses the try, except, else and finally blocks to manage exceptions.

🛡 try...except Example

```python try: num = int(input("Enter a number: ")) print(100 / num) except ZeroDivisionError: print("Cannot divide by zero.") except ValueError: print("Invalid input.") ```

🔄 else and finally

```python try: print("File opened successfully.") except: print("An error occurred.") else: print("No exception occurred.") finally: print("Program Finished.") ```

📊 Common Exceptions

Exception Description
ZeroDivisionError Occurs when dividing by zero.
ValueError Occurs when an invalid value is provided.
TypeError Occurs when incompatible data types are used.
FileNotFoundError Occurs when the specified file does not exist.
IndexError Occurs when accessing an invalid list index.
KeyError Occurs when a dictionary key is not found.

🌍 Applications

  • ✔ Reading and writing student records.
  • ✔ Managing employee databases.
  • ✔ Logging application data.
  • ✔ Saving user information.
  • ✔ Error handling in software applications.
  • ✔ Data backup and report generation.

⭐ Advantages

  • ✔ Prevents unexpected program crashes.
  • ✔ Improves application reliability.
  • ✔ Makes file management easy.
  • ✔ Supports secure error handling.
  • ✔ Enhances user experience.

  • ❌ Incorrect file handling may cause data loss.
