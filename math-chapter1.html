📘 Chapter 1

Relation, Sets & Functions

This chapter introduces the basic concepts of Sets, Relations and Functions. These topics are fundamental in Mathematics and Computer Science and are widely used in databases, programming, networking and data structures.

📚 Beginner ⏱ 2 Hours ⭐ Important 📝 Exam Topic

📖 Introduction

A Set is a collection of well-defined objects. Relations describe the relationship between two sets, while Functions represent a special type of relation where every input has exactly one output.

🎯 Learning Outcomes

  • Understand Sets and their representation.
  • Perform Set Operations.
  • Understand Cartesian Product.
  • Learn Relations and their Types.
  • Study Functions and Types of Functions.
  • Applications in Computer Science.

📚 1. Sets

A Set is a well-defined collection of distinct objects.

A = {1,2,3,4,5}

Examples

A = {2,4,6,8}

B = {Apple, Mango, Orange}

Methods of Representation

  • Roster Method
  • Set Builder Method

Types of Sets

  • Empty Set
  • Singleton Set
  • Finite Set
  • Infinite Set
  • Universal Set
  • Subset
  • Proper Subset
  • Equivalent Set

📚 2. Set Operations

Set Operations are used to combine or compare two or more sets.

1. Union of Sets

A ∪ B = {Elements present in A or B}
A = {1,2,3}
B = {3,4,5}

A ∪ B = {1,2,3,4,5}

2. Intersection of Sets

A ∩ B = {Common Elements}
A ∩ B = {3}

3. Difference of Sets

A − B = Elements of A not present in B

4. Complement of Set

A' = Universal Set − A

📚 3. Cartesian Product

The Cartesian Product of two sets A and B is the set of all ordered pairs.

A × B
A = {1,2}
B = {a,b}

A × B = {(1,a),(1,b),(2,a),(2,b)}

📚 4. Relations

A Relation is a subset of Cartesian Product.

R ⊆ A × B

Types of Relations

  • Reflexive Relation
  • Symmetric Relation
  • Transitive Relation
  • Equivalence Relation

Applications

  • Database Management System
  • Graph Theory
  • Networking
  • Artificial Intelligence

📚 5. Functions

A Function is a special type of relation in which every input has exactly one output.

f : A → B

Types of Functions

  • One-One Function
  • Many-One Function
  • Onto Function
  • Into Function
  • Bijective Function

Applications

  • Programming
  • Computer Graphics
  • Machine Learning
  • Artificial Intelligence
  • Data Science

📝 Important Formulae

n(A ∪ B) = n(A) + n(B) − n(A ∩ B)
Number of Subsets = 2ⁿ
A × B = {(a,b) | a ∈ A and b ∈ B}
R ⊆ A × B
f : A → B

📌 Chapter Summary

  • Set is a collection of well-defined objects.
  • Sets can be represented using Roster and Set Builder methods.
  • Union combines all elements.
  • Intersection gives common elements.
  • Cartesian Product forms ordered pairs.
  • Relation is a subset of Cartesian Product.
  • Function is a special type of Relation.
  • Functions are widely used in Computer Science.
