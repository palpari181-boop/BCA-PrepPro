Chapter 1: Set Theory (Complete Detailed Notes)

A set is a well-defined collection of distinct objects. These objects are called elements of the set. A set is usually represented using capital letters like A, B, C etc. Example: A = {1, 2, 3, 4, 5}

Representation of Sets:
1. Roster Form: A = {1,2,3,4}
2. Set Builder Form: A = {x : x is natural number less than 5}

Types of Sets:
• Empty Set (∅): No elements present
• Finite Set: Limited number of elements
• Infinite Set: Unlimited elements
• Singleton Set: Only one element
• Equal Sets: Same elements
• Subset: A ⊆ B means all elements of A are in B

Universal Set: A set that contains all possible elements under discussion.

Operations on Sets:
• Union (A ∪ B): All elements in A and B
• Intersection (A ∩ B): Common elements
• Difference (A − B): Elements in A but not in B
• Complement (A’): Elements not in A (in universal set)

Laws of Sets:
• Commutative Law: A ∪ B = B ∪ A
• Associative Law: (A ∪ B) ∪ C = A ∪ (B ∪ C)
• Distributive Law: A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)

De Morgan’s Laws:
(A ∪ B)’ = A’ ∩ B’
(A ∩ B)’ = A’ ∪ B’

Chapter 2: Relations & Functions (Detailed Notes)

A relation is a subset of Cartesian product of two sets. If A and B are two sets then R ⊆ A × B.

Example: A = {1,2}, B = {3,4} R = {(1,3), (2,4)}

Types of Relations:
• Reflexive: (a,a) must exist for all a ∈ A
• Symmetric: If (a,b) ∈ R then (b,a) ∈ R
• Transitive: If (a,b) and (b,c) then (a,c)
• Equivalence Relation: Reflexive + Symmetric + Transitive

Function:
A function is a relation in which every input has exactly one output. Notation: f(x) = y

Types of Functions:
• One-One (Injective): Different inputs → different outputs
• Onto (Surjective): Every output has input
• Into: Some outputs are unused
• Bijective: Both One-One and Onto

Important Terms:
Domain: Set of input values
Codomain: Possible output values
Range: Actual output values

Chapter 3: Matrices (Complete Detailed Notes)

A matrix is a rectangular arrangement of numbers, symbols, or expressions arranged in rows and columns. If a matrix has m rows and n columns, it is called an m × n matrix.

Example: A = [ 1 2 3 4 5 6 ] This is a 2 × 3 matrix.

Notation: A = [aᵢⱼ] where i = row number and j = column number

Types of Matrices:
• Row Matrix: Only one row
• Column Matrix: Only one column
• Square Matrix: Same rows and columns (n × n)
• Rectangular Matrix: m × n where m ≠ n
• Zero Matrix: All elements are 0
• Diagonal Matrix: Non-diagonal elements are 0
• Identity Matrix (I): Diagonal elements = 1

Matrix Operations:
1. Addition: Only same order matrices can be added
2. Subtraction: Same rule as addition
3. Scalar Multiplication: Multiply each element by constant
4. Matrix Multiplication: Row × Column rule

Important Rule of Multiplication:
If A is m×n and B is n×p then AB is possible and result is m×p matrix.

Transpose of Matrix:
Transpose of A (Aᵀ) is obtained by converting rows into columns.

Example: A = [1 2 3 4 5 6] Aᵀ = [1 4 2 5 3 6]

Symmetric Matrix: Aᵀ = A
Skew Symmetric Matrix: Aᵀ = -A

Inverse of Matrix:
A⁻¹ exists only if determinant of A is not zero (det A ≠ 0)

Formula: A⁻¹ = adj(A) / det(A)

Chapter 4: Determinants (Complete Detailed Notes)

A determinant is a scalar value associated with a square matrix. It is denoted as |A|.

Determinant of 2×2 Matrix:
|a b| |c d| = ad − bc

Determinant of 3×3 Matrix:
It is expanded using minors and cofactors.

General form: |A| = a₁₁M₁₁ − a₁₂M₁₂ + a₁₃M₁₃

Minors: Determinant of sub-matrix formed by deleting row and column
Cofactors: Signed minors

Properties of Determinants:
• If two rows are identical, determinant = 0
• If a row is zero, determinant = 0
• Interchanging rows changes sign
• Multiplying a row multiplies determinant
• det(A) = det(Aᵀ)

Important Formulas:
det(AB) = det(A) × det(B)

Cramer’s Rule:
Used to solve linear equations: Ax = B
Solution: x = Dx / D, y = Dy / D, z = Dz / D

Applications:
• Solving linear equations
• Finding area of triangle
• Engineering calculations

Chapter 5: Probability (Complete Detailed Notes)

Probability is a branch of mathematics that deals with the chance or likelihood of occurrence of an event. It is used to measure uncertainty.

Basic Formula:
P(E) = Number of favourable outcomes / Total number of outcomes

Probability values always lie between: 0 ≤ P(E) ≤ 1

• If P(E) = 0 → impossible event
• If P(E) = 1 → sure event

Important Terms:
• Experiment: Any process that produces outcomes
• Sample Space (S): Set of all possible outcomes
• Event (E): Subset of sample space

Example:
Throwing a dice:
S = {1,2,3,4,5,6}

Types of Events:
• Simple Event: One outcome
• Compound Event: More than one outcome
• Sure Event: Always occurs
• Impossible Event: Never occurs

Equally Likely Events:
Events having same probability of occurrence. Example: Tossing a fair coin (H or T)

Mutually Exclusive Events:
Two events that cannot happen at the same time. A ∩ B = 0

Independent Events:
Occurrence of one event does not affect another.

Addition Theorem of Probability:
P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

If A and B are mutually exclusive:
P(A ∪ B) = P(A) + P(B)

Conditional Probability:
Probability of A given B has already occurred.

Formula:
P(A|B) = P(A ∩ B) / P(B)

Multiplication Theorem:
P(A ∩ B) = P(A) × P(B|A)

If A and B are independent:
P(A ∩ B) = P(A) × P(B)

Bayes Theorem (Basic Idea):
Used to reverse conditional probabilities in advanced problems.

Important Applications:
• Weather prediction
• Risk analysis
• Insurance calculations
• Game theory

Expected Value (Extra Important):
E(X) = Σ [x × P(x)]

This helps in decision making under uncertainty.
